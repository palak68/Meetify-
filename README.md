# Meetify

Meetify is a web-based meeting and collaboration platform that allows users to create and join online meetings easily. It helps users collaborate remotely with features like authentication, meeting creation, real-time chat, and screen sharing.

## Features

- User Registration
- Secure Login and Logout
- Protected Routes using Authentication
- User Dashboard
- Create Meetings
- Join Meetings using a Meeting Link
- Real-time Chat during meetings
- Screen Sharing for presentations and collaboration
- Responsive and user-friendly UI

## Project Workflow

1. User registers on the platform  
2. User logs in using email and password  
3. After login, user is redirected to the dashboard  
4. User creates a meeting  
5. A unique meeting link is generated  
6. Other users join the meeting using the link  
7. Participants can chat in real time during the meeting  
8. Users can share their screen for presentations
